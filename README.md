# Week 10 ADK: Agent Development Kit

โปรเจกต์นี้เป็นการพัฒนา AI Agent ด้วยกรอบการทำงาน Agent Development Kit (ADK) โดยทำหน้าที่เป็นผู้เชี่ยวชาญด้านวิทยาศาสตร์ข้อมูล (Data Scientist) ในการดึงและวิเคราะห์ชุดข้อมูลจาก UCI Machine Learning Repository

## 🌟 ความสามารถหลัก (Features)
- ดึงข้อมูล Metadata ของชุดข้อมูลจาก UCI ML Repository โดยใช้ ID ของชุดข้อมูล
- สรุปรายละเอียดและวิเคราะห์ข้อมูลเบื้องต้นสำหรับงาน Data Science
- ขับเคลื่อนด้วยโมเดล `gemini-2.0-flash`
